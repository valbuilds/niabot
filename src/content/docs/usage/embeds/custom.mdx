title: Custom Embeds

import { Card } from '@astrojs/starlight/components';

To make custom embeds, head to [message.style](https://message.style/) and log in.

Make your embeds with the visualiser, then click "JSON Code" at the top.

Copy and paste the code into a new file under the `config/embeds` folder. Make sure it ends in `.json`.

<Card title="Congrats!">You've made a custom embed to use with the bot.</Card>
